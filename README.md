# Vidhima Construction Pvt Ltd — Website

Business website for **Vidhima Construction Private Limited**, a Class-I registered construction company based in Palampur, Kangra, Himachal Pradesh.

## Business

Three verticals:
- **Government Tenders** — roads, buildings, bridges across HP
- **Real Estate Construction** — turnkey residential & commercial builds on client plots
- **Sub-Contracting** — civil work packages for larger contractors

## Tech stack

- Vite + React + TypeScript
- Tailwind CSS + shadcn/ui
- Hosted on Netlify
